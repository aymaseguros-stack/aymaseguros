# Contribución a AYMA Advisors

## Estructura del Proyecto
```
src/
├── components/
│   ├── Header.jsx       # Navegación + logo
│   ├── HeroSection.jsx  # Formulario cotización
│   ├── ChatBot.jsx      # Bot conversacional
│   ├── Sections.jsx     # WhyChooseUs, Services, etc.
│   └── Footer.jsx       # Contacto + mapas
├── App.jsx              # Componente principal
└── main.jsx             # Entry point
```

## Comandos
```bash
npm run dev      # Desarrollo local
npm run build    # Build producción
npm run preview  # Preview build
npm test         # Ejecutar tests
```

## Convenciones

- Commits: `feat:`, `fix:`, `docs:`, `style:`, `refactor:`
- Componentes: PascalCase
- Archivos: kebab-case o PascalCase para componentes
